# Prompt de Redesign — Front-end (FitBlock / Coach Híbrido)

## Objetivo
Redesenhar o front-end com estética premium (tech/elite), mantendo performance e clareza para 3 áreas:
1. Home Page
2. Login / Entrada no Web App
3. Dashboard do Treinador
4. Dashboard do Aluno/Atleta

Inspirado em:
- Interface tática (OATH style)
- Minimalismo premium (Apple + dark systems)
- UX rápida para treino

---

# Identidade Visual

## Cores
- Primária: #41226b (roxo marca)
- Secundária: #121212 (dark base)
- Neutra: #f1f0e9 (background clean)
- Destaque: #FF3300 (ação / energia)

## Tipografia
- Títulos: Eurostile Extended (ou similar)
- Texto: Montserrat
- Mono: para dados (tempo, score, métricas)
